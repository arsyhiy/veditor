* Обработка ввода пользователя

Редактор должен поддерживать событийный цикл.

#+BEGIN_SRC c
while (1) {
    int c = getch();

    switch (c) {
        case KEY_LEFT:
            move_cursor_left();
            break;

        case KEY_RIGHT:
            move_cursor_right();
            break;

        case KEY_UP:
            move_cursor_up();
            break;

        case KEY_DOWN:
            move_cursor_down();
            break;

        case KEY_BACKSPACE:
            delete_char();
            break;

        default:
            insert_char(c);
            break;
    }

    refresh_screen();
}
#+END_SRC

* Отрисовка экрана

Рекомендуется использовать dirty redraw стратегию.

Алгоритм:

1. Очистить экран
2. Отрисовать видимую часть текста
3. Обновить курсор

Минимизировать мерцание экрана.
